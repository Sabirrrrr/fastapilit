1. Backend Teknolojileri:
   - FastAPI: Modern, hızlı web framework
   - SQLAlchemy: Python SQL toolkit ve ORM
   - Pydantic: Veri doğrulama ve serialization
   - JWT: JSON Web Token tabanlı kimlik doğrulama
   - PostgreSQL: İlişkisel veritabanı sistemi
   - Uvicorn: ASGI web sunucusu

2. Frontend Core Teknolojileri:
   - TypeScript: Tip güvenli JavaScript süper kümesi
   - Lit: Hafif, performanslı web component framework
   - Tailwind CSS: Utility-first CSS framework
   - Vite: Modern frontend build tool

3. Frontend Geliştirme Araçları:
   - Node.js: JavaScript runtime
   - npm/yarn: Paket yöneticisi
   - ESLint: TypeScript/JavaScript linter
   - Prettier: Kod formatlayıcı
   - TypeScript Compiler: TS -> JS dönüşümü

4. Web Components (Lit ile Geliştirilecek):
   - Auth Components:
     * Login Form
     * Register Form
     * User Profile
   - Post Components:
     * Post List
     * Post Card
     * Post Editor
     * Post Details
   - Vote Components:
     * Vote Button
     * Vote Counter
   - Common Components:
     * Navigation Bar
     * Modal
     * Toast Notifications
     * Loading Spinner

5. API İletişim Katmanı:
   - Fetch API / Axios
   - TypeScript interfaces (Backend şemalarıyla uyumlu)
   - JWT token yönetimi
   - Error handling

6. State Yönetimi:
   - Lit reactive properties
   - Custom events
   - Browser Local Storage

7. Routing:
   - Client-side routing
   - Protected routes
   - Route guards

8. Stil ve Tema:
   - Tailwind konfigürasyonu
   - Custom CSS variables
   - Responsive tasarım
   - Dark/Light tema desteği
